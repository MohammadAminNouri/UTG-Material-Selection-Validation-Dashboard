# Report-ready text: GitHub, Streamlit, and ML validation-support section

A GitHub repository and Streamlit dashboard were developed as a digital companion to the material-selection report. The purpose of the tool is not to replace Granta EduPack, ANSYS, or physical UTG testing, but to make the material-selection logic reproducible and easier to inspect. The dashboard includes the five shortlisted candidate materials, the Ashby performance indices, the weighted decision matrix, a sensitivity analysis of selection weights, a bending-stress and chemical-strengthening check, an Eco Audit sensitivity page, and a structured validation-plan checklist.

The computational module recalculates the main indices used in the report: flexural strength divided by Young's modulus for elastic foldability, fracture toughness divided by Young's modulus for displacement-controlled flaw tolerance, fracture toughness squared divided by Young's modulus for fracture-energy resistance, hardness divided by density for surface durability per unit mass, and thermal expansion multiplied by Young's modulus for thermal-mismatch tendency. These calculations reproduce the selection logic that led to Alumino silicate 1720 as the preferred candidate before final product validation.

A small machine-learning decision-support module was also added. The model uses the material-property table and the calculated Ashby indices as inputs, while the weighted decision score from the report is used as the target output. Therefore, the ML result should be interpreted only as a surrogate model for checking the consistency and sensitivity of the decision matrix. It is not a direct predictor of fold-cycle lifetime, fracture probability, or commercial UTG reliability, because the available dataset contains only the shortlisted materials and does not include experimental fatigue data.

The validation page of the dashboard separates completed and future validation work. The completed validation is the preliminary ANSYS folding simulation, which showed maximum tensile stress at the fold region. This supports the selection logic because it confirms that the hinge/fold line is the critical mechanical zone. However, the simulation is not treated as a final pass/fail proof. Final approval of Alumino silicate 1720 as processed foldable UTG still requires chemical strengthening validation, surface-compressive-stress measurement, edge-quality inspection, static bend-radius testing, cyclic folding fatigue tests, scratch and hardness tests, pen-drop or indentation testing, optical inspection, lamination testing, and manufacturing-yield assessment.
